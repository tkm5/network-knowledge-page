---

ネットワーク関連の主要な専門用語をまとめています．各用語の詳細は関連ページを参照してください．

## A

- **ACK（Acknowledgement）**: 受信確認．TCPでデータを正しく受信したことを送信側に通知するメッセージ → [第6章 TCP](/network-knowledge-page/chapter06/04-tcp/)
- **ARP（Address Resolution Protocol）**: IPアドレスからMACアドレスを解決するプロトコル → [第5章 ARP](/network-knowledge-page/chapter05/03-arp/)
- **AS（Autonomous System）**: 自律システム．単一の管理組織によって運営されるネットワークの集合 → [第7章 ルーティング範囲](/network-knowledge-page/chapter07/02-routing-scope/)

## B

- **BGP（Border Gateway Protocol）**: AS間の経路制御プロトコル → [第7章 BGP](/network-knowledge-page/chapter07/06-bgp/)
- **ブリッジ**: データリンク層（レイヤ2）でフレームを中継する装置 → [第1章 ネットワーク構成要素](/network-knowledge-page/chapter01/09-network-components/)
- **ブロードキャスト**: 同一ネットワーク内の全ホストに一斉にデータを送信する通信方式 → [第1章 通信方式](/network-knowledge-page/chapter01/07-communication-types/)

## C

- **CIDR（Classless Inter-Domain Routing）**: クラスレスのIPアドレス割り当て方式 → [第4章 IPアドレス](/network-knowledge-page/chapter04/03-ip-address/)
- **CSMA/CD**: イーサネットで使用される媒体アクセス制御方式 → [第3章 データリンク技術](/network-knowledge-page/chapter03/02-datalink-technology/)

## D

- **DHCP（Dynamic Host Configuration Protocol）**: IPアドレスの自動割り当てプロトコル → [第5章 DHCP](/network-knowledge-page/chapter05/05-dhcp/)
- **DNS（Domain Name System）**: ドメイン名とIPアドレスを相互変換するシステム → [第5章 DNS](/network-knowledge-page/chapter05/02-dns/)

## E

- **Ethernet（イーサネット）**: 最も広く使われている有線LAN規格 → [第3章 イーサネット](/network-knowledge-page/chapter03/03-ethernet/)

## F

- **FTP（File Transfer Protocol）**: ファイル転送プロトコル → [第8章 FTP](/network-knowledge-page/chapter08/03-ftp/)
- **ファイアウォール**: ネットワーク間のトラフィックを制御するセキュリティ機能 → [第9章 セキュリティ構成要素](/network-knowledge-page/chapter09/02-security-components/)

## G

- **ゲートウェイ**: 異なるプロトコルのネットワーク間を中継する装置 → [第1章 ネットワーク構成要素](/network-knowledge-page/chapter01/09-network-components/)

## H

- **HTTP（HyperText Transfer Protocol）**: Webページの転送に使用されるプロトコル → [第8章 WWW](/network-knowledge-page/chapter08/05-www/)

## I

- **ICMP（Internet Control Message Protocol）**: IPの補助プロトコル．エラー通知や診断に使用 → [第5章 ICMP](/network-knowledge-page/chapter05/04-icmp/)
- **IP（Internet Protocol）**: インターネット層の中核プロトコル → [第4章 IP](/network-knowledge-page/chapter04/01-ip-overview/)
- **IPv6**: IPの次世代バージョン．128ビットアドレス空間を持つ → [第4章 IPv6](/network-knowledge-page/chapter04/06-ipv6/)
- **IPsec**: IP層でのセキュリティプロトコル → [第9章 セキュリティプロトコル](/network-knowledge-page/chapter09/04-security-protocols/)

## L

- **LAN（Local Area Network）**: 限定された範囲のネットワーク → [第1章 ネットワーク基礎](/network-knowledge-page/chapter01/01-background/)

## M

- **MACアドレス**: データリンク層で使用される48ビットの物理アドレス → [第3章 データリンク技術](/network-knowledge-page/chapter03/02-datalink-technology/)
- **MPLS（Multi-Protocol Label Switching）**: ラベルベースのパケット転送技術 → [第7章 MPLS](/network-knowledge-page/chapter07/07-mpls/)
- **MTU（Maximum Transmission Unit）**: データリンクで送信可能な最大データサイズ → [第4章 分割処理](/network-knowledge-page/chapter04/05-fragmentation/)

## N

- **NAT（Network Address Translator）**: プライベートIPとグローバルIPの変換技術 → [第5章 NAT](/network-knowledge-page/chapter05/06-nat/)

## O

- **OSI参照モデル**: 通信プロトコルの7階層モデル → [第1章 OSI参照モデル](/network-knowledge-page/chapter01/05-osi-model/)
- **OSPF（Open Shortest Path First）**: リンク状態型のIGP → [第7章 OSPF](/network-knowledge-page/chapter07/05-ospf/)

## P

- **PPP（Point-to-Point Protocol）**: 2点間通信のデータリンクプロトコル → [第3章 PPP](/network-knowledge-page/chapter03/05-ppp/)
- **ポート番号**: トランスポート層でアプリケーションを識別する番号 → [第6章 ポート番号](/network-knowledge-page/chapter06/02-port-number/)
- **プロトコル**: 通信を行うための規約 → [第1章 プロトコル](/network-knowledge-page/chapter01/03-protocol/)

## Q

- **QUIC（Quick UDP Internet Connections）**: UDP上で動作する高速トランスポートプロトコル → [第6章 その他のトランスポート](/network-knowledge-page/chapter06/05-other-transport/)

## R

- **RFC（Request for Comments）**: インターネット標準文書 → [第2章 TCP/IP標準化](/network-knowledge-page/chapter02/02-tcpip-standardization/)
- **RIP（Routing Information Protocol）**: 距離ベクトル型のルーティングプロトコル → [第7章 RIP](/network-knowledge-page/chapter07/04-rip/)
- **ルーター**: ネットワーク層でパケットを中継する装置 → [第1章 ネットワーク構成要素](/network-knowledge-page/chapter01/09-network-components/)

## S

- **SMTP（Simple Mail Transfer Protocol）**: メール送信プロトコル → [第8章 電子メール](/network-knowledge-page/chapter08/04-email/)
- **SNMP（Simple Network Management Protocol）**: ネットワーク管理プロトコル → [第8章 SNMP](/network-knowledge-page/chapter08/06-snmp/)
- **SSH（Secure Shell）**: セキュアな遠隔ログインプロトコル → [第8章 SSH](/network-knowledge-page/chapter08/02-telnet-ssh/)
- **サブネットマスク**: IPアドレスのネットワーク部とホスト部を識別するビットマスク → [第4章 IPアドレス](/network-knowledge-page/chapter04/03-ip-address/)

## T

- **TCP（Transmission Control Protocol）**: 信頼性のあるコネクション型トランスポートプロトコル → [第6章 TCP](/network-knowledge-page/chapter06/04-tcp/)
- **TLS/SSL**: トランスポート層のセキュリティプロトコル → [第9章 セキュリティプロトコル](/network-knowledge-page/chapter09/04-security-protocols/)

## U

- **UDP（User Datagram Protocol）**: 軽量なコネクションレス型トランスポートプロトコル → [第6章 UDP](/network-knowledge-page/chapter06/03-udp/)

## V

- **VLAN（Virtual LAN）**: 仮想的に分割されたLAN → [第3章 データリンク技術](/network-knowledge-page/chapter03/02-datalink-technology/)
- **VPN（Virtual Private Network）**: 仮想プライベートネットワーク → [第3章 公衆アクセス網](/network-knowledge-page/chapter03/07-public-access/)

## W

- **Wi-Fi**: IEEE802.11規格に基づく無線LAN → [第3章 無線通信](/network-knowledge-page/chapter03/04-wireless/)
